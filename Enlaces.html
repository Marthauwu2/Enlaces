Ejemplo de Enlaces con Atributos

A continuación, se muestran ejemplos de enlaces con diferentes atributos:

Atributo href

El atributo href define el destino del enlace. En este caso, dirige a una pagina sobre la cultura del antiguo egipto.

Cultura Egipcia

Atributo download

Indica que el enlace es para descargar un archivo. Aquí descarga un archivo PDF sobre reglas del ajedrez.

Reglas Ajedrez

Atributo hreflang

El atributo hreflang indica el idioma del contenido al que enlaza. En este caso, dirige a una página en inglés.

Abrir Pagina en inglés

Atributo rel

Define la relación entre el documento actual y el recurso vinculado. Este enlace utiliza rel="nofollow".

Enlace Pinterest

Atributo target

Especifica dónde se abrirá el documento vinculado. Aquí se abre en una nueva pestaña (_blank).

Abrir video en una nueva pestaña

Atributo type

Especifica el tipo de medio del recurso vinculado. Aquí se define como un archivo de imagen PNG.

Descargar imagen PNG
